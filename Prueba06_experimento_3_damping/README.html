Prueba 6 experimento 3 damping

El damping provoca un cambio directamente en la velocidad de un objeto. Funciona con la siguiente ecuación:

Se va a tomar el código anterior y se va a modificar, primero se eliminan las variables e inicialización de las listas. Así como todo lo relacionado con mesh y shader.

Se cambia el nombre de las variables, F por Vi y X por V. La velocidad inicial será de 10. Se agrega la variable damping con un valor de 0.9.

```javascript private float Vi=10,t=0,V=10,damping=.9f; ```

Se cambia el nombre de las variables pos1, pos2, por vel1, vel2.

```javascript private String str,vel1,vel2,error; ```

Se modifica el ancho de la pantalla a 8m.

```javascript An=8; ```

Se modifica el cubo, su fricción y restitución van a ser 0, se cambia la posición inicial, se aplica la velocidad inicial y el valor de damping.

```javascript PolygonShape Cushape=new PolygonShape(); BodyCu=world.createBody(createBody(-5f,0,'D')); Cushape.setAsBox(.5f,.5f); CuFix=BodyCu.createFixture(createFix(Cushape,1,0,0)); Cushape.dispose(); BodyCu.setLinearVelocity(Vi,0); BodyCu.setLinearDamping(damping); ```

En la función render, se limpia la pantalla y se crea el formato de texto.

```javascript Gdx.gl.glClear(GL20.GL_COLOR_BUFFER_BIT); Gdx.gl.glClearColor(0.1f,0f,0.1f,0.5f); DecimalFormat df = new DecimalFormat("#0.00"); ```

Se usa la ecuación de la velocidad afectada por el damping.

```javascript V=V*(1-delta*damping); ```

Se guardan los valores de velocidad, así como el error.

```javascript vel1=df.format(BodyCu.getLinearVelocity().x); vel2=df.format(V); error=df.format(Math.abs(BodyCu.getLinearVelocity().x-V)); ```

Se cambia lo que se va a mostrar en pantalla, y se corrige e nombre de las variables.

```javascript batch.begin(); font.draw(batch,"Velocidad cuerpo ="+vel1,Box2Pix(-2),Boy2Piy(2.2f)); font.draw(batch,"Velocidad ecuacion="+vel2,Box2Pix(-2),Boy2Piy(1.7f)); font.draw(batch," error ="+error,Box2Pix(-2),Boy2Piy(1.2f)); batch.end(); ```
