Prueba 7 experimento 4 voltear cubo

La fuerza normal que recibe un cuerpo se puede encontrar en un punto que este en contacto con una superficie. Si al realizar un cálculo de la distancia a la que se encuentra la normal, esta se encuentra fuera del cuerpo, implica que el objeto se va a voltearo está siendo volteado. Un objeto comienza a voltearse cuando la distancia X es igual a cero.

Se usará un modelo estático, con una fuerza (F) aplicada en el centro con una normal (N) a una distancia X. El cubo tendrá 2m por lado.

Para obtener el valor de X se va a realizar la sumatoria de pares de torsión en la esquina del cubo.

Se va a usar el código de la prueba de fricción (estático).

La variable velocidad se va a cambiar por posición y V por P.

```javascript private String Fuerza,posicion; private float F=0,P=0; ```

Se va a cambiar la creación del cubo. Va a tener 2m de lado.

```javascript PolygonShape Cushape=new PolygonShape(); BodyCu=world.createBody(createBody(-2,.5f,'D')); Cushape.setAsBox(1,1); ```

Para evitar que avance mientras se le aplica la fuerza, se aumentará la fricción.

```javascript CuFix=BodyCu.createFixture(createFix(Cushape,1,100,0)); ```

En la función render en lugar de obtener la velocidad se va a obtener la posición de X.

```javascript P=((10*BodyCu.getMass()-F)*2)/(10*BodyCu.getMass()); ```

Convertir a String la posición

```javascript posicion=df.format(P); ```

Mostrar en pantalla.

```javascript batch.begin(); font.draw(batch,"Fuerza = "+Fuerza,Box2Pix(1),Boy2Piy(1.8f)); font.draw(batch,"Posicion = "+posicion,Box2Pix(-2),Boy2Piy(1.8f)); batch.end(); ```

Se va a cambiar la condición de la fuerza, va a dejar de aumentar cuando X llegue a 0.

```javascript if(P>=0) { F=F+.05f; } ```
